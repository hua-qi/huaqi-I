## 使用方式

### 1. 初始化记忆

```bash
# 交互式向导（推荐）
huaqi memory init

# 快速初始化
huaqi memory init --quick --name "你的名字" --occupation "职业"
```

### 2. 查看记忆

```bash
# 查看所有记忆
huaqi memory list

# 搜索记忆
huaqi memory search "电吉他"

# 查看记忆库状态
huaqi memory status
```

### 3. 手动编辑

你可以直接用任何 Markdown 编辑器打开 `~/.huaqi/users_data/{user_id}/memory/` 下的文件进行编辑。
