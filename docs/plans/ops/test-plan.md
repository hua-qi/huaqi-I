## 2. 测试目标

本文档为 Code Agent 提供完整的测试执行指南，确保重构后的系统满足：

1. **功能完整性**：所有 PRD 功能正常工作
2. **架构正确性**：LangGraph 架构、状态管理、人机协同机制正确
3. **数据安全**：迁移过程不丢数据，可回滚
4. **性能达标**：响应延迟、检索速度符合要求

|------|---------|

| 人机协同无法恢复 | 检查 checkpoint 持久化 |
