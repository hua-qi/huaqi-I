# 长期模式学习系统

从用户的日记、对话和动态维度中学习长期行为模式，生成个性化的周报和洞察。

## 功能概述

模式学习系统通过分析用户的长期数据，识别情绪周期、话题偏好和行为模式，帮助用户更好地了解自己，并提供个性化的成长建议。

## 识别模式类型

### 1. 情绪周期

识别用户情绪的周期性规律：

- **星期几模式**：周几情绪最低/最高
- **情绪波动性**：情绪稳定度分析
- **长期趋势**：情绪整体改善或恶化趋势

### 2. 话题趋势

追踪用户关注话题的变化：

- **持续关注话题**：反复提及的主题
- **话题演变**：兴趣点的转移
- **新兴话题**：新出现的关注点

### 3. 行为模式

分析用户的行为习惯：

- **活跃时段**：一天中什么时候最活跃
- **交互频率**：对话频率的变化
- **记录习惯**：日记记录的规律性

## 周报内容

每周自动生成报告，包含：

```
📊 本周洞察 (2024-03-25 ~ 2024-03-31)

情绪概览：情绪平稳（平均 5.5/10）
情绪趋势：➡️ 平稳 / ↗️ 上升 / ↘️ 下降
热门话题：工作(8次)、健康(5次)、职业规划(3次)

💡 发现
  • 周一情绪低落：你的情绪在周一通常较低...
  • 情绪持续改善：最近的情绪比上半月提升了...

🎯 建议
  • 这周情绪有些下滑，周末可以安排一些放松活动
  • 情绪波动较大，试试每天写3件感恩的事
```

## 使用方式

### 命令

```bash
# 查看最新周报（自动生成或手动触发）
/report

# 手动生成本周报告
/report weekly

# 查看模式洞察列表
/report insights
```

### 自动触发

每周首次启动 CLI 时，系统会自动检查并生成本周报告：

```
$ huaqi chat

🌸 Huaqi - 个人 AI 同伴

📊 本周洞察 (2024-03-25 ~ 2024-03-31)
...
```

## 洞察类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 情绪周期 | 每周固定时间的情绪规律 | "周一情绪低落" |
| 成长轨迹 | 情绪的长期变化趋势 | "情绪持续改善" |
| 话题趋势 | 持续关注的话题 | "持续关注：职业发展" |
| 行为模式 | 交互行为的变化 | "日记频率提升" |

### 洞察严重程度

| 级别 | 图标 | 说明 |
|------|------|------|
| POSITIVE | 🟢 | 积极发现 |
| INFO | 🔵 | 信息性 |
| WARNING | 🟡 | 需要关注 |
| ATTENTION | 🔴 | 重要提醒 |

## 数据需求

模式学习需要一定的数据量才能生成可靠的洞察：

| 分析类型 | 最小数据量 | 理想数据量 |
|---------|-----------|-----------|
| 周报生成 | 3 篇日记 + 5 条对话 | 7 篇日记 + 20 条对话 |
| 情绪周期 | 7 天数据 | 30 天数据 |
| 话题趋势 | 10 条对话 | 50 条对话 |
| 长期趋势 | 14 天数据 | 90 天数据 |

## 数据存储

### 周报存储
- 路径：`~/.huaqi/memory/weekly_reports.jsonl`
- 格式：JSON Lines，每行一条周报记录

### 洞察存储
- 路径：`~/.huaqi/memory/pattern_insights.jsonl`
- 格式：JSON Lines，包含洞察详情和有效期

### 洞察有效期

洞察有一定的有效期（默认30天），过期后需要重新验证：
- 情绪周期：30 天
- 话题趋势：14 天
- 行为模式：30 天

## 隐私说明

- 所有模式分析在本地完成
- 用户可以查看、删除自己的模式数据
- 周报和洞察仅用户本人可见
