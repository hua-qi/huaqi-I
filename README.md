# Huaqi - 个人 AI 同伴系统

> 不是使用 AI，而是养育 AI —— 让它越来越像你，越来越懂你

## 核心理念

**Human 3.0**: 通过 AI 基础设施实现人类能力的大规模增强。

**关系定位**: 不是主仆，而是**同伴 (Peer)** —— 共同成长的数字伙伴。

**设计原则**: 
- Humans set direction, AI executes at scale
- 模块化、可进化、可迁移
- 隐私优先、本地优先、用户可控

## 功能特性

- **💬 智能对话**: 支持多行输入，AI 基于你的画像、日记、历史对话回答
- **📝 日记系统**: 记录每日心情与想法，Markdown 格式，支持批量导入
- **🎯 技能追踪**: 添加技能、记录练习时间、追踪成长进度
- **🎖️ 目标管理**: 设定短期/长期目标，追踪完成进度
- **💾 对话历史**: 自动保存每次对话，支持搜索回顾
- **👤 个性引擎**: 可配置的 AI 性格（沟通风格、共情水平、幽默程度）
- **🔒 隐私优先**: 所有数据本地存储，用户完全可控

## 快速开始

```bash
# 安装
pip install huaqi

# 初始化
huaqi config init

# 开始对话
huaqi chat
```
