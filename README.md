# Huaqi - 个人 AI 同伴系统

> 不是使用 AI，而是养育 AI —— 让它越来越像你，越来越懂你

## 核心理念

**Human 3.0**: 通过 AI 基础设施实现人类能力的大规模增强。

**关系定位**: 不是主仆，而是**同伴 (Peer)** —— 共同成长的数字伙伴。

**设计原则**: 
- Humans set direction, AI executes at scale
- 模块化、可进化、可迁移
- 隐私优先、本地优先、用户可控

## 功能特性

- **记忆系统**: 三层记忆架构（会话/工作/长期），混合存储方案
- **个性引擎**: 可配置的 AI 性格，越用越懂你
- **技能生态**: 基于 MCP 协议的可扩展技能系统
- **自动化**: Hook 系统实现事件驱动的工作流
- **跨设备同步**: Git 原生支持，数据随身携带

## 快速开始

```bash
# 安装
pip install huaqi

# 初始化
huaqi config init

# 开始对话
huaqi chat
```
